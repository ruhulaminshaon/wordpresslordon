Vendor wizard setup is a intergration plugin for Dokan of WP. It help easy setting vendor account.
It included:
1) Back-end: Include config APP-URL
2) Front-end: An beautifully step and setting by steps
All other URLs will render by just one menu item
Note: It is working with hight STYLE with WP base config. All just query to this one shortcode [vendor_setup_wizard_main] and open back-end to save it

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress 
3. Check shortcodes to create views and use it
